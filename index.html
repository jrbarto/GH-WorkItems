GH Work Items

A system to create and track work items against GitHub organizations.
How It Works
work

GH Work Items maintains a database filled with all organizations and repositories of those organizations of users who have registered an account. Any active user may access our work item system and file a ticket against repositories managed by our system. Members of the organizations can login and view tickets filed against their repositories.

Getting Started
done

New users may click the register button at the top of the page to create a new account. Please note that you must have a GitHub account in order to use this system to track your organizations. If you do not have a GitHub account, you may visit GitHub to create an account.
